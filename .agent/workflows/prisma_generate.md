---
description: Generate Prisma client
---
1. Ensure you are in the project root (`e:\et-ticket-app\services\api`).
2. Run the Prisma generate command:
   ```
   npx prisma generate
   ```
3. Verify that the client was regenerated without errors.
4. Restart the development server if it was running.
5. Re-run your TypeScript build or `npm run dev` to see the updated types.
